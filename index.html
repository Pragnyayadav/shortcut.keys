List of All Shortcut Keys in Windows
with Usage


    Alt+F4

    It is used to turn off any software or whole computer.

    Ctrl+W

    It is used to turn off a tab of a browser like chrome, edge, firefox etc.

    Ctrl+C

    It is used for copying the selected text or any other objects. The text/object will be copied to the Clipboard.

    Ctrl+X

    It is used to cut selected text and save it to clipboard

    Ctrl+V

    It is used for pasting the text/object which is copied to clipboard. This is mostly used with Ctrl + C or Ctrl + x.

    Ctrl + Shift + Esc

    It is used to open Task Manager. Task Manager is used to control all the running tasks and many other things.

    Ctrl+S

    It is used to Save. It is used to save any file on which you are working on MS Excel, Powerpoint ect. or to save a webpage in a browser.

    Ctrl+O

    It is used to open a URL, document, image, or other file types. Mostly used in Excle, Word etc.

    Alt+F

    It is used to open file menu options in the current program.

    Shift+Del

    It is used to delete selected item.

    Alt+E

    Open Edit options in current program.

    Alt+Tab

    Switch between open programs.

    F1

    Opens help information of open program or Windows

    F2

    Rename a selected file

    F5

    Refresh the current program.

    Ctrl+D

    Bookmarks the current page in most browsers.

    Ctrl+A

    Selects all text part in a software or file.

    Ctrl+B

    Change the selected text to bold.

    Ctrl+U

    Underline the selected text.

    Ctrl+I

    Change the selected text to italics.

    Ctrl+F

    Open find window for the current open software or document.

    Ctrl+Instert

    Copy the selected text.

    Shift+Insert

    Paste the text or image present in clipboard

    Ctrl+Y

    Redo the lest action.

    Ctrl+Z

    Undo the last action.
